# backend

Structure:
* routes - Controllers
* lib - Libraries
* public - Static files

###### Build: ```npm run  build```   
###### Deploy: ```npm run deploy ```   
###### Start development: ```npm run start```   
###### Apply migration: ```npm run migrate <migration_name>```    

About migration:
  * It must be SQL file, that contains SQL table creation code
  * migration_name must be without extension
